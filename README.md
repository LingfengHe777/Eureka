《Eureka》		类型：2D 类幸存者肉鸽		Unity游戏客户端独立开发

展示地址： https://www.bilibili.com/video/BV1jQo1BeE14

技术栈：Unity3D、C#、UGUI、Animator、Addressables、ScriptableObject、对象池

1. UI 界面系统：独立完成首页菜单、局内 HUD、升级奖励、属性栏、商店、结算面板等界面的搭建与交互逻辑开发。

2. PVE GamePlay玩法：实现类幸存者 PVE 核心循环，包括波次推进、可选难度、无尽模式、敌人持续生成、敌人追击与攻击。

3. 升级与随机奖励系统：

·	实现玩家经验升级、升级奖励选择、属性成长和波间强化流程

·	升级奖励和商店商品通过 ScriptableObject 配置管理，支持不同武器、道具、属性效果的扩展

4. 商店与构筑系统：

·	实现随机商品刷新、锁定、购买、出售回收等功能

·	武器系统支持槽位管理、购买添加、同名同等级武器二合一合成升级

·	道具系统支持配置化效果、层数叠加和移除回退

5. 资源与性能优化：

·	使用 Addressables 管理角色、敌人、武器、道具、UI 等资源加载

·	实现对象池复用敌人、子弹、金币、生成预警和战斗飘字，减少频繁 Instantiate/Destroy 带来的性能开销

6. 项目亮点：

·	独立完成从 0 到 1 的 UGUI 界面搭建和交互逻辑开发，并实现完整 PVE 玩法闭环

·	项目采用 ScriptableObject 配置化管理敌人、波次、武器、道具、模式和数值策略，提升后续扩展与调参效率

·	实现了相机平滑跟随、目标绑定、偏移控制等功能，增强局内操作体验
